## -------------------------------------------------------------------------
## Xenia Back-end Description

[NestJs] The back-end application for xenia assignment.

## Installation
```bash
$ cd xenia-be
$ yarn install
```

## Running the app
```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```
